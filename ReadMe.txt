In this repository you will find all the source code for the project and all of the confusion matrices 
generated from the validation part. 

All the data has been removed  to ensure no confidential data is published.

main.py:
This file is the main file where all of the parameters of the program can be set.

loadfile.py:
This file handles all the loading and separation of the data. 

ML.py:
This file contains all the manual implementation of PCA, SVD-LDA, MLP and SMO-SVM.

SL-Model.py:
This file simply enables for the MLP weights and biases to be loaded and saved.
